Algolia JS Helper

The toolbox for creating advanced search experiences

Made for building your next advanced search experience with Algolia

The helper is built on top of the Algolia JS Client. It provides all the necessary tools to make your next search experience. With advanced faceting features, your users will find what they’re looking for even faster and precisely than ever before.

Get Started

Rock solid foundation

The Helper is opinionated on the structure of the search logic but not about the context on which it is used. The Helper defines:

  • An immutable model that is the source of truth for the search parameters
  • Events to dispatch the steps of the search flow
Learn more

Any framework friendly

The Helper is all the frameworks best friend. Its architecture doesn’t have any ties with a specific framework nor it leaks abstractions that are foreign to your tools. We even used it for creating our own widget library instantsearch.js
